# 业务理解

- 软件名称：ZDEM盐体运动学分析系统
- 产品定位：面向构造地质学与盐构造研究领域，专为ZDEM离散元模拟数据设计的盐体运动学自动化分析系统。从数万颗粒的原始模拟数据中自动提取盐丘几何形态演化指标，生成符合学术出版标准的诊断图表。
- 面向领域 / 行业：构造地质学/盐构造研究/地球科学
- 核心价值：替代传统人工逐帧测量方式，实现盐体运动学指标的自动化批量提取与可视化，大幅提升构造缩短模拟数据的分析效率和分析结果的一致性。
- 证据文件：`业务理解证据.md`

## 目标用户

- 构造地质学研究人员
- 盐构造与盐丘演化研究学者
- 地球物理学与盆地模拟研究人员
- 石油地质与油气勘探工程技术人员

## 主要业务功能

- 批量数据提取
- 智能特征点定位
- 交互式质控修正
- 学术级诊断绘图
- 多实验组联合比对

## 功能说明

- 批量数据提取：系统按实验组批量解析.dat格式的离散元粒子数据文件，通过水平分箱(binned_statistic)提取模型表面包络线，结合峰值检测(find_peaks)和坑底追踪算法定位盐丘主峰与边缘向斜基点，计算宽度、起伏、高宽比三大运动学指标，结果存储为CSV表格和PKL缓存文件。
- 智能特征点定位：采用坑底探测+反弹判定(Rio Tinto)算法，从峰顶向推板反方向扫描，追踪局部极小值点，当连续反弹超过阈值时锁定基点位。辅以局部斜率阈值作为终止条件，确保边缘向斜边界识别的准确性。
- 交互式质控修正：系统逐帧展示盐体粒子分布与算法识别的特征点叠加图。用户通过鼠标左键点击盐体表面可重新定位基点，右键撤销上一步操作，按S键或Enter键保存修正结果并自动进入下一帧。
- 学术级诊断绘图：空间维采用自适应窗口Savitzky-Golay滤波器平滑包络线，时间维采用中心化滚动平均滤波演化曲线。图表输出采用Y轴右置学术风格、出露前实线+出露后虚线标识、600 DPI高精度渲染。
- 多实验组联合比对：基于ProcessPoolExecutor多进程并行解析各组数据，自动汇聚所有实验组的运动学演化轨迹，生成PNG位图与PDF矢量图两种格式的联合对比图，支持自定义组标签、颜色与标记样式。

## 典型操作流程

1. 配置环境变量：复制.env.example为.env，设置数据目录路径和实验组配置
2. 运行数据提取：执行01_data_extractor.py，批量解析.dat文件，生成CSV指标表和PKL缓存
3. 人工质控修正：执行01b_manual_corrector.py，交互式检查并修正特征点定位
4. 单组诊断出图：执行02_plot_single_diagnostic.py，生成演化曲线和剖面诊断图
5. 多组联合对比：执行03_plot_multi_comparison.py，生成跨实验组对比图

## 操作手册页面/流程模块

1. 环境配置：用户在终端中进入软件根目录，首先执行pip install -r requirements.txt安装依赖，然后将.env.example复制为.env并编辑。
   - 使用场景：用户首次使用软件时需要配置运行环境。用户在终端中执行安装命令，然后编辑配置文件指定数据目录路径和实验组信息。
   - 操作要点：在终端中进入软件根目录；执行pip install -r requirements.txt安装科学计算依赖；将模板文件.env.example复制为.env；使用文本编辑器打开.env文件，修改FINAL_OUTPUT_DIR为实际数据存放路径
2. 数据提取：用户在终端中执行python 01_data_extractor.py命令启动数据提取过程。
   - 使用场景：用户完成环境配置后，需要从原始模拟数据中提取盐体运动学指标。执行本步骤后，软件自动读取所有实验组的数据文件，计算每帧的盐体宽度、起伏和高宽比。
   - 操作要点：在终端中执行python 01_data_extractor.py命令；软件自动遍历.env中配置的所有实验组；对每个实验组的data目录，按文件名编号顺序读取所有.dat文件；软件解析粒子坐标数据，通过分箱统计提取表面包络线
3. 交互式质控修正：用户在终端中执行python 01b_manual_corrector.py命令，软件弹出Matplotlib交互窗口。
   - 使用场景：数据提取完成后，研究者需要检查算法自动识别的基点是否准确。如果某些帧的基点定位有偏差，用户可以通过质控界面手动修正。
   - 操作要点：执行python 01b_manual_corrector.py启动交互界面；软件自动加载第一帧的数据和算法识别结果；在图形窗口中查看盐体粒子分布和特征点标注；如需修正基点：鼠标左键点击盐体表面任意位置，基点自动更新为点击坐标
4. 单组诊断绘图：用户在终端中执行python 02_plot_single_diagnostic.py命令。
   - 使用场景：数据提取和质控完成后，研究者需要生成可视化图表用于分析盐体演化规律和论文发表。
   - 操作要点：执行python 02_plot_single_diagnostic.py命令；软件读取每个实验组的kinematics_data.csv数据；生成运动学演化轨迹图（Kinematic_Evolution_Diagnostic.png），展示该组的高宽比随时间变化；生成盐体剖面多宫格诊断图（Salt_Profiles_Diagnostic_Grid.png），以多宫格形式展示不同时刻步的盐体轮廓和特征点
5. 多组联合比对：用户在终端中执行python 03_plot_multi_comparison.py命令。
   - 使用场景：当用户有多个实验组（如不同缩短速率、不同盐层厚度等参数对比）时，需要将各组结果放到一起比较，观察参数变化对盐体演化的影响。
   - 操作要点：执行python 03_plot_multi_comparison.py命令；软件自动加载所有实验组的kinematics_data.csv数据；在同一坐标系中绘制各组的高宽比演化曲线，使用不同颜色和标记区分；在同一坐标系中绘制各组的起伏演化曲线

## 申请表建议口径

- 开发目的：为构造地质学与盐构造研究提供盐体运动学指标的自动化定量分析工具，替代传统人工逐帧测量的低效方式
- 软件的主要功能：本软件是面向构造地质学与盐构造研究的盐体运动学自动化分析工具。用户将ZDEM离散元模拟的.dat格式粒子数据按实验组整理后，软件可批量解析所有时刻步的原始数据，自动提取盐丘剖面的几何形态演化指标。软件核心处理流程分为四个步骤：第一步，数据提取——通过水平分箱统计与自适应Savitzky-Golay平滑，从数万颗粒中提取模型表面包络线，应用峰值检测和坑底追踪算法定位盐丘主峰与边缘向斜基点，计算宽度、起伏和高宽比三大运动学指标，结果存储为CSV表格和PKL缓存文件；第二步，交互式质控——通过Matplotlib交互界面逐帧展示算法识别结果，用户可鼠标点击修正算法误判的基点位置，系统自动保存修正记录；第三步，单组诊断——为每个实验组生成600 DPI的运动学演化轨迹图和盐体剖面多宫格诊断图；第四步，多组比对——自动汇聚所有实验组数据，生成宽度、起伏、高宽比的联合对比图，同时输出PNG和PDF矢量格式。
- 技术特点：基于Python科学计算生态(NumPy/SciPy/Pandas/Matplotlib)，采用分箱统计与自适应Savitzky-Golay滤波算法，融合坑底探测与反弹判定机制精准定位边界基点，支持多进程并行处理大规模实验组，输出600 DPI学术级图表

- 软件分类：应用软件
